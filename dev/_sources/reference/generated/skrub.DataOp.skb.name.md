# skrub.DataOp.skb.name

#### DataOp.skb.name

A user-chosen name for the DataOp.

The name is used for display, to retrieve a specific node inside the
DataOp or to override its value. See [`DataOp.skb.set_name()`](skrub.DataOp.skb.set_namehtml.md#skrub.DataOp.skb.set_name) for
more information.
