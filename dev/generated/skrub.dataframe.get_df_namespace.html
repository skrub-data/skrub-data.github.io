skrub.dataframe.get_df_namespace#

skrub.dataframe.get_df_namespace(*dfs)[source]#

Get the namespaces of dataframes.

Introspects dataframes and returns their skrub namespace object skrub.dataframe._{pandas, polars} and the dataframe module {polars, pandas} itself.

The dataframes passed in input need to come from the same module, otherwise a TypeError will be raised.

The outputs of this function are denoted skrub_px and px in reference to the array API, returning namespace (NumPy, PyTorch and CuPy) as nx. Since we deal with Polars (pl) and Pandas (pd), we use px as a variable name.

Parameters:
dfsDataFrameLike | list[DataFrameLike],

The dataframes to extract modules from.

Returns:
skrub_pxModuleType

Skrub namespace shared by dataframe objects.

pxModuleType

Dataframe namespace, i.e. Pandas or Polars module.
